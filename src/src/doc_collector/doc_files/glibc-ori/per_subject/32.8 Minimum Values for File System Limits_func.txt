- _POSIX_MAX_CANON  
The most restrictive limit permitted by POSIX for the maximum number of bytes in a canonical input line from a terminal device.  The value of this constant is 255.

- _POSIX_MAX_INPUT  
The most restrictive limit permitted by POSIX for the maximum number of bytes in a terminal device input queue (or typeahead buffer). See Input Modes.  The value of this constant is 255.
